# tic-tac-toe



A Python-based Tic-Tac-Toe game featuring both Normal and Advanced gameplay modes. The project provides a simple and interactive gaming experience, while the advanced version introduces more challenging game logic and AI-based computer moves. It demonstrates the practical implementation of Python programming, conditional logic, functions, loops, and game development concepts.

Author: Pijush Pakrashi
Programming Language: Python
Project Type: Game Development
Version: Normal & Advanced AI Mode
